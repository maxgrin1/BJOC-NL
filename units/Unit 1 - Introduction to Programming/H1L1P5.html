Greeting the Player

Jouw klik op Alonzo script wordt uitgevoerd when Alonzo is clicked. Op deze pagina, maak je een tweede script dat de speler verwelkomd wanneer er op de groene vlag () wordt geklikt.

Blokken die eruit zien als deze worden gebruikt om een gebeurtenis zoals het klikken op de sprite, het indrukken van een toets, of het klikken van de groene vlag te detecteren. Ze vertellen het script dat het moet uitvoeren wanneer de gebeurtenis plaatsvindt. Omdat deze blokken zich alleen bovenaan het script kunnen bevinden (als een manier om het script te activeren), noemen Snap! programmeurs ze "kop blokken."

Klikken op de groene vlag () zorgt ervoor dat alle scrips die beginnen met wanneer de groene vlag is geklikt kop blok, geactiveerd worden. Je kan alle actieve scrips doen laten stoppen wanneer je op het rode stopteken() rechts boven het Snap! tabblad klikt.

  1. Als hij nog niet open is, open het U1L1-ClickAlonzo bestand.
    2. Je kan dit doen met slechts 3 blokken met o.a :
  2. Maak een tweede script dat Alonzo beweegt naar (190, -130) en zeg een welkomst boodschap wanneer de groene vlag wordt geklikt. (Alonzo kan bijvoorbeeld zeggen, "Welkom bij 'het Klik Op Alonzo Spel!' blijf op mij klikken als je kan!") .
  3. Zorg ervoor dat de boodschap weggaat wanneer de speler op Alonzo klikt.
    The block with an empty input will make the sprite say nothing.
  4. Controleer of je App doet wat je wilt:
    • Wanneer er op de groene vlag geklikt wordt, zou Alonzo moeten bewegen naar (190, -130) en de boodschap roepen.
    • Wanneer er op Alonzo geklikt wordt, zou de boodschap moeten verdwijnen en zou Alonzo naar een willekeurige plaats op het podium moeten bewegen en de andere kant op wijzen.
